Plataformas


WepSIM funciona en cualquier plataforma que use alguno de los navegadores Web más populares:
Google Chrome 80+, Mozilla Firefox 74+, Microsoft Edge 85+, Apple Safari 12+

Una versión para Android 5.0+ está también disponible en la tienda de Google Play.

Tecnologías


WepSIM usa diferentes tecnologías como:

  • Frameworks: Bootstrap Vue, Jquery,
  • Iconos: font awesome, icon-icons
  • Plug-ins y extensiones: Bootbox, Editor CodeMirror, Spectrum ColorPicker, Biblioteca de visualización vis.js, Jquery Knob
