Platforms


WepSIM works in any platform that uses any of the most popular modern Web Browsers:
Google Chrome 50+, Mozilla Firefox 45+, Microsoft Edge 30+, and Apple Safari 10+

An Android 5.0+ version version is also available at Google Play.

Technologies


WepSIM uses different technologies such as:

  • Frameworks: Jquery Mobile, Knockout, Bootstrap
  • Icons: glyphicons, icon-icons
  • Plug-ins & extensions: Bootbox, CodeMirror Editor, Spectrum ColorPicker, Bootstrap Notify, vis.js visualization library, Jquery Knob
