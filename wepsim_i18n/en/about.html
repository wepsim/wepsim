Platforms


WepSIM works in any platform that uses any of the most popular modern Web Browsers:
Google Chrome 80+, Mozilla Firefox 74+, Microsoft Edge 85+, Apple Safari 12+

An Android 5.0+ version version is also available at Google Play.

Technologies


WepSIM uses different technologies such as:

  • Frameworks: Bootstrap Vue, Jquery,
  • Icons: font awesome, icon-icons
  • Plug-ins & extensions: Bootbox, CodeMirror Editor, Spectrum ColorPicker, vis.js visualization library, Jquery Knob
